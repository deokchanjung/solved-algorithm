# 십진수를 이진수로 변환하기

## 문제

십진수를 이진수로 변환하여 출력하기

## 풀이

```javascript
const decimalToBinary = n => {
  let binStr = '';

  const decimalToBinaryHelper = n => {
    if (n < 2) {
      binStr += n;
      return;
    } else {
      binStr += decimalToBinaryHelper();
    }
  };
};
```
